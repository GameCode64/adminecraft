# AD//MIN//CRAFT
## A php based minecraft server control backend

## Requirements:
 - PHP 8.2+
 - Composer 2.7.1+

## Installation:
Run:
```sh cp .env.example .env && composer install && php artisan migrate:fresh && php artisan key:generate```
